ℹ️ Dropbox setup

1. Go to dropbox.com/developers → App Console
2. Select your app (or create: Scoped access, Full Dropbox)
3. Permissions tab: tick files.metadata.read + files.content.read, Submit
4. Copy your App Key from Settings tab
5. Paste App Key above, set folder path
6. Tap Connect Dropbox, authorize, copy code, paste & submit
7. Done! Token auto-refreshes. Tap Import All New
